# Pong Game

Este es un mini-juego de **Pong** hecho en Python usando la biblioteca `turtle`. Forma parte de una colección de mini-juegos que estaré desarrollando para mi portafolio.

## Descripción

El juego es una versión simple del clásico Pong:
- Controla dos paletas para golpear la pelota.
- Mantiene el puntaje para cada jugador.
- Incluye sonido de rebote (solo en Windows).

## Controles

- **Jugador A**: `W` (arriba), `S` (abajo)
- **Jugador B**: Flecha `Arriba` (arriba), Flecha `Abajo` (abajo)

## Requisitos

- Python 3
- Biblioteca "turtle"
- Archivo de sonido `bounce.wav` para el efecto de rebote

## Instalación

1. Clona el repositorio y navega a la carpeta del proyecto.
2. Ejecuta el script:

   ```bash
   python pong.py
